# Aufgabe

Gegeben sind zwei ganze Zahlen $a$ und $b$. Berechne die Summe dieser beiden Zahlen.

# Eingabedaten

Die erste Zeile enthält zwei Zahlen $a$ und $b$.

# Ausgabedaten

Die erste Zeile enthält eine einzelne Zahl, die die Summe der beiden Zahlen darstellt.

# Einschränkungen und Hinweise

* $1 \leq a, b \leq 1 \ 000 \ 000$;
* Dies ist eine Beispielaufgabenstellung. Sie kann beliebig angepasst werden, wobei das Format möglichst beibehalten werden sollte. Eine Anleitung mit weiteren Informationen zu den Formatierungsrichtlinien für Aufgabenstellungen wird in Zukunft veröffentlicht.

# Beispiel 1

`{{.InputFile}}`
``
1 2
``

`{{.OutputFile}}`
``
3
``

## Erklärung

Wie man sieht, ist $1 + 2 = 3$.
